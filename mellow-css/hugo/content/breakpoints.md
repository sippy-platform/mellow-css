# Breakpoints
Breakpoints determine at which points Mellow changes the behavior of any breakpoint-related class.

| Breakpoint | Suffix | Pixels |
| ---------- | ------ | ------ |
| Extra Small | - | < 600px |
| Small | `sm` | >= 600px |
| Medium | `md` | >= 800px |
| Large | `lg` | >= 1000px |
| Extra Large | `xl` | >= 1200px |
| Extra Extra Large | `xxl` | >= 1400px |
