Features

Built-in colors

Changing the colors of components is as simple as changing a single class. No matter the component.

Extensive components

Mellow comes with an extensive set of components, from buttons, to progress, to toatsts. Cards? Dropdowns? We've got them.

Built-in themes

Mellow comes with built-in support for light and dark themes, and supports following the system theme too. Change the theme of your browser and OS, and see how Mellow changes this page.
