# Dialog
The Dialog component is similar to the Dialog component in its behavior and API.

<Canvas>
  <Story name="Dialog">
    {() => {
      const [open, setOpen] = useState(false);
      return (
        <>
          <Button onClick={() => setOpen(true)}>Open dialog</Button>
          <Dialog open={open} setOpen={setOpen}>
            <DialogHeader action={<CloseButton setOpen={setOpen} />}>Dialog example</DialogHeader>
            <DialogBody>
              <p>
                Are you sure you want to deactivate your account? All of your data
                will be permanently removed. This action cannot be undone.
              </p>
            </DialogBody>
          </Dialog>
        </>
      );
    }}
  </Story>
</Canvas>

## Disable backdrop
If you don't want a backdrop to be rendered, that's possible too.

<Canvas>
  <Story name="Backdropless dialog">
    {() => {
      const [open, setOpen] = useState(false);
      return (
        <>
          <Button onClick={() => setOpen(true)}>Open dialog without Backdrop</Button>
          <Dialog open={open} setOpen={setOpen} disableBackdrop>
            <DialogHeader action={<CloseButton setOpen={setOpen} />}>Dialog example</DialogHeader>
            <DialogBody>
              <p>
                Are you sure you want to deactivate your account? All of your data
                will be permanently removed. This action cannot be undone.
              </p>
            </DialogBody>
          </Dialog>
        </>
      );
    }}
  </Story>
</Canvas>
