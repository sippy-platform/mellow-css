# Label
The button is one of our simplest components.

## Example
A basic button can be created with only a label.

<Canvas>
  <Story
    name="Default"
    args={{
      children: 'Default',
      color: 'red'
    }}
  >
    {Template.bind({})}
  </Story>
</Canvas>

## Plated
Set `plated` as the `variant` to get a plated label.

<Canvas>
  <Story
    name="Plated variant"
    args={{
      children: 'Red',
      variant: 'plated',
      color: 'red'
    }}
  >
    {Template.bind({})}
  </Story>
</Canvas>

## Solid
Set `solid` as the `variant` to get a solid label.

<Canvas>
  <Story
    name="Solid variant"
    args={{

      color: 'red'
    }}
  >
    {Template.bind({})}
  </Story>
</Canvas>
