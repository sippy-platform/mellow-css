# Grid
The grid is your primary tool to build out the layout of your app. Mellow's grid provides various powerful tools to help you do this.

## Grid columns
By default, the grid will create a CSS grid with 12 columns. Every element inserted as a child will take on 1 column if no other classes are given.

You can also make the column count respond to breakpoints.
