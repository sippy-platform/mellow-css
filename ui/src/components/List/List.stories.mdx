## Dividers
There are 2 variants of dividers: `full` and `default`.

### Default
You can enable the default dividers by simply adding the `dividers` prop.

<Canvas>
  <Story
    name="Default dividers"
    args={{
      border: true,
      dividers: true,
      children: ([
        <ListItem primary="This is a list item" />,
        <ListItem primary="This is also list item" />
      ])
    }}
  >
    {Template.bind({})}
  </Story>
</Canvas>

### Full
The full dividers can be enabled by setting `dividers` to `full`.

<Canvas>
  <Story
    name="Full dividers"
    args={{
      border: true,
      dividers: 'full',
      children: ([
        <ListItem primary="This is a list item" />,
        <ListItem primary="This is also list item" />
      ])
    }}
  >
    {Template.bind({})}
  </Story>
</Canvas>

### Individual
Use the `ListDivider` to add dividers for individual objects.

<Canvas>
  <Story
    name="Individual dividers"
    args={{
      border: true,
      children: ([
        <ListItem primary="This is a list item" />,
        <ListItem primary="This is also list item" />,
        <ListDivider />,
        <ListItem primary="This is a list item" />,
        <ListItem primary="This is also list item" />,
        <ListDivider full />,
        <ListItem primary="This is a list item" />,
        <ListItem primary="This is also list item" />
      ])
    }}
  >
    {Template.bind({})}
  </Story>
</Canvas>

### Section dividers
With the `ListItemDivider` component you can split up your list into sections.

<Canvas>
  <Story
    name="Section dividers"
    args={{
      border: true,
      dividers: true,
      children: ([
        <ListItemDivider primary="Section divider" />,
        <ListItem primary="This is a list item" />,
        <ListItem primary="This is also list item" />,
        <ListItemDivider primary="Section divider" secondary="Section description" />,
        <ListItem primary="This is also list item" />,
        <ListItem primary="This is also list item" />
      ])
    }}
  >
    {Template.bind({})}
  </Story>
</Canvas>

## Color
You can set the color of the list globally or for each item.

<Canvas>
  <Story
    name="Colored list"
    args={{
      color: 'green',
      children: ([
        <ListItem primary="This list item inherits green" active />,
        <ListItem primary="This list item inherits green too" active />,
        <ListItem primary="This one wants to be edgy" color="red" active />

    }}
  >
    {Template.bind({})}
  </Story>

### Full color
By adding the `fullColor` prop, the list item will theme the entire item.

<Canvas>
  <Story
    name="Full colored list"
    args={{
      color: 'green',

        <ListItem primary="This is the default look" onClick={() => alert('You clicked')} />,

  >

</Canvas>
