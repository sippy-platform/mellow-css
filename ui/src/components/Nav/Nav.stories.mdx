# Nav
A basic navigation menu consists out of the `Nav` component with nested `NavGroup` components and `NavItem` components within these groups.

<Canvas>
  <Story
    name="Nav"
    args={{
      children: ([
        <NavGroup className="me-auto">
          <NavItem href="#">Home</NavItem>
          <NavItem href="#">About</NavItem>
        </NavGroup>,
        <NavGroup>
          <NavItem href="#">Settings</NavItem>
        </NavGroup>
      ])
    }}
  >
    {Template.bind({})}
  </Story>
</Canvas>
