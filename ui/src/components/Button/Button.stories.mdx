import { ArgsTable, Canvas, Meta, Story } from '@storybook/addon-docs';

export const Template = (args) => <Button {...args} />;

# Button
The button is one of our simplest components.

## Example
A basic button can be created with only a label.

<Canvas>
  <Story
    name="Default"
    args={{
      children: 'Default'
    }}
  >
    {Template.bind({})}
  </Story>
</Canvas>

## Links
If you want your button to be an anchor, simply include a `href` tag and Mellow UI takes care of the rest.

<Canvas>
  <Story
    name="Link"
    args={{
      children: 'Link',
      href: 'https://sippy.cloud'
    }}
  >
    {Template.bind({})}
  </Story>
</Canvas>

## Colored
Buttons can take on any of our main palette colors. Apply the `color` variant to use them.

<Canvas>
  <Story
    name="Color"
    args={{
      children: 'Red',
      variant: 'color',
      color: 'red'
    }}
  >
    {Template.bind({})}
  </Story>
</Canvas>

## Color on hover
Need a colored button that draws less attention? The colored-on-hover button is perfect for this purpose.

<Canvas>
  <Story
    name="Color on hover"
    args={{
      children: 'Red',
      variant: 'hover',
      color: 'red'
    }}
  >
    {Template.bind({})}
  </Story>
</Canvas>

## Semantic buttons
For your ease and conviniance, Mellow maps a number of variants to a special semantically named type. Note that the `color` option has no effect on any of the semantic button variants.

    }}
  >
    {Template.bind({})}
  </Story>
