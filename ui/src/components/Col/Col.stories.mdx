# Columns
The `Col` component is ideal for when you have to manipulate the default column behavior. However, note that the grid is designed to work fine with any component as a direct child. You may not need the `Col` component.

## Column spans
You can change columns to span more of the grid with the breakpoint properties.

## Order
Change the order of the columns with the order prop.

<Canvas>
  <Story
    name="Order"
    args={{
      xs: 5,
      children: ([

        <Col xs={{ order: 1 }} className="bg-cyan p-2 rounded text-light">2</Col>,
        <Col xs={{ order: 3 }} className="bg-teal p-2 rounded text-light">3</Col>,

  >
    {Template.bind({})}
  </Story>

### First and last
Aside from a range between 0 and 5, you can also use `first` and `last`.

<Canvas>
  <Story
    name="First and last"
    args={{
      xs: 5,
      children: ([
        <Col xs={{ order: 0 }} className="bg-blue p-2 rounded text-light">1</Col>,
        <Col xs={{ order: "last" }} className="bg-cyan p-2 rounded text-light">2</Col>,
        <Col xs={{ order: "first" }} className="bg-teal p-2 rounded text-light">3</Col>,
        <Col xs={{ order: 2 }} className="bg-green p-2 rounded text-light">4</Col>,
        <Col xs={{ order: 4 }} className="bg-lime p-2 rounded text-light">5</Col>
      ])
    }}
  >
    {Template.bind({})}
  </Story>
</Canvas>

### First and last
**Note**: by default, grid items behave as if `order: 0` has been set to them. This means that the order-utility classes only work relative to eachother. For example, setting order to 2 to the third item and no order to any other column will result in the 3rd item being moved to the end of the line.

<Canvas>
  <Story
    name="Order default"
    args={{
      xs: 5,
      children: ([
        <Col className="bg-blue p-2 rounded text-light">1</Col>,
        <Col className="bg-cyan p-2 rounded text-light">2</Col>,
        <Col xs={{ order: 2 }} className="bg-teal p-2 rounded text-light">3</Col>,
        <Col className="bg-green p-2 rounded text-light">4</Col>,
        <Col className="bg-lime p-2 rounded text-light">5</Col>
      ])
    }}
  >
    {Template.bind({})}
  </Story>
</Canvas>
