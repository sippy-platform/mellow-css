# Radio group
When using the Radio component, it must be used within a RadioGroup component. The RadioGroup component controls the state for the other components.

<Canvas>
  <Story
    name="Basic radio"
    args={{
      name: 'basic',
      value: 'hello',
      loop: true,
      required: false,
      children: ([
        <Radio value="hello" label="Hello" id="hello" />,
        <Radio value="bonjour" label="Bonjour" id="bonjour" />
      ])
    }}
  >
    {Template.bind({})}
  </Story>
</Canvas>

## With helper
Add a helper prop to add extra information.

  <Story

  >

## No label
If, for any reason, you ever find yourself in a situation where a radio button doesn't need a label, you can leave it out.

      ])
    }}

    {Template.bind({})}
